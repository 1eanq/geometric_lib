# Описание решения
Вычисления производятся с помощью следующих формул:

## Площадь
- Круг: S = πR²
- Прямоугольник: S = ab
- квадрат: S = a²

## Периметр
- Круг: P = 2πR
- Прямоугольник: P = 2a + 2b
